Getting Started

  • Two devices open the same Remoteboard (that is, provide the same "Remoteboard ID");
  • Draw on one device. You should see (in some inevitable delay) on another device!
  • Make sure both devices have good Internet connection; otherwise it may not be as "realtime" as you may expect.
  • If you don't see it on another device after 30 seconds, on the second device,
    • Refresh the board by opening it again;
    • Closes the app and reopen it.
    • Check, in Settings -> Remoteboard, that push notifications are allowed.
    • If the problem persists, contact administrator by adding a pull request to This repository on Github

快速开始

  • 两部设备通过输入相同的白板ID来打开同一个远程白板。
  • 在一部设备上画画，你会实时在另一台设备上看到结果！（可能稍有延迟）
  • 保证网络畅通，否则可能会有显著延迟。
  • 如果30秒后仍然没有在另一台设备上看到笔画，在没有看到笔画的设备上，
    • 重新打开白板（菜单中重新选择“打开白板”然后输入相同的ID）；
    • 重启App；
    • 在系统设置->远程白板中检查推送通知是否被允许。
    • 如果问题仍未解决，请在这个Github项目上增加一条Pull Request。

FAQ

  • Q: I want to share it on Weibo / Facebook / Twitter etc., but did not found such options in the Share menu.
    A: Go to System Settings -> Facebook (for example) -> log in with your account. This should fix the issue.

常见问题

  • 问：我想在微博 / Facebook / Twitter 等分享，但并没有在分享菜单中看到选项。
    答：请查看系统设置->新浪微博（举个例子）->登录您的账号，即可解决。
